# SSR with Loadable Components

## Setup

1. Install dependencies:
   ```sh
   npm install
   ```
2. Build client and server:
   ```sh
   npm run build
   ```
3. Start the SSR server:
   ```sh
   npm run start:ssr
   ```

Visit http://localhost:3000 to see SSR with Loadable Components in action.
